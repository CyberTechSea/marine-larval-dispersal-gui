🌊 Marine Larval Dispersal — Help

WoRMS Species Search

  • Type any scientific name — the interface queries WoRMS REST API in real time
  • The traits API (marinespecies.org/traits/) is queried for larval development data
  • Traits are pre-filled when available in WoRMS literature; edit freely when absent
  • PLD is rarely stored as a numeric value in WoRMS — enter it from your own data if needed

Replicates — Temporal Design

  • Choose the temporal unit (hours, days, weeks, months, seasons, years)
  • Each replicate is one simulation run with a specific start date
  • Use presets for common designs (monthly Jan–Jun, 4 seasons, etc.)
  • For hourly simulations, ensure your CMEMS data has hourly resolution

File Manager

  • Click 📁 (header or map panel) to open the file manager
  • Download figures as PNG, SVG, or PDF
  • Download CSV statistics for external analysis
  • All files are also saved to the output directory

Quick Start

  1. Start backend: conda activate sim_env && python backend/server.py
  2. Set CMEMS file path (Data tab or export NC_FILE=…)
  3. Search species in WoRMS (Species tab)
  4. Add release sites (Sites tab)
  5. Configure replicates (Replicates tab)
  6. Click Run Simulation
